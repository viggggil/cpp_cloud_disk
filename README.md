# TinyWebServer-Like Scaffold

This project is a starter scaffold inspired by TinyWebServer.

## Structure

- `CGImysql/`: database pool abstractions
- `http/`: HTTP connection handling
- `lock/`: synchronization wrappers
- `log/`: logging abstraction
- `threadpool/`: worker thread pool
- `timer/`: timer management
- `root/`: static web root
- `test_pressure/`: pressure testing notes/scripts

## Build

```bash
sh ./build.sh
```

## Run

```bash
./server
```
